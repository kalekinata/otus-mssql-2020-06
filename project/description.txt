База данных для железнодорожного вокзала(Train_Station).
Необходимо выполнение возможности выбора пассажиром удобного для него маршрута, по которому будет следовать поезд; удобного для него времени и даты; тип вагона и класс обслуживания; место, на котором он будет располагаться на протяжение всего пути.
В соответствии с этим необходимо наличие таблиц, где указываются:
- маршруты (routes);
- характеристики поезда (trains);
- информация о вагонах (cars);
- данные о пассажирах (passengers);
- остановки (stopping);
- описание типов вагонов (cars_type) и моделей поездов (trains_model). 
- таблицы с информацией, содержащейся в билетах (ticket);
- расписание с указанием маршрутов, поездов и дат (timetable);
- таблица с указынием статуса билета, то есть оплачен или нет (pay);
- таблица, содержащая информацию об остановках (routes_stop);
- таблица, содержащая информацию о составе поезда (composition).
